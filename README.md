# Jerseyworld
![JersyWorld](logo.jpeg)


# Technologies
- NextJs(typescript)
- Strapi
- Prisma

# Description

# Starting Strapi Server
```shell
    cd ./backend
    yarn run develop`
```
# Starting NextJs Server
```
    cd ./frontend
    yarn
    yarn install
    yarn run dev
```
